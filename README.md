# API Sail

> Projeto Laravel com Sail para ambiente de desenvolvimento simplificado.

---

## 🚀 Instalação

Execute os comandos abaixo para instalar as dependências e preparar o ambiente:

```bash
composer install
npm install
npm run build
sail artisan migrate
```

---

## 💻 Acesso

Acesse o projeto em: [http://localhost/](http://localhost/)
